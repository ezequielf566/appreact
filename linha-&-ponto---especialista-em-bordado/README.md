# Linha e Ponto - Especialista em Bordado

Este é um projeto React + Vite para transformar fotos em riscos de bordado.

## Como rodar localmente

1. `npm install`
2. `npm run dev`

## Deploy no Vercel

1. Conecte seu repositório ao Vercel.
2. Adicione a variável de ambiente `VITE_GEMINI_API_KEY`.
3. O Vercel deve detectar automaticamente como um projeto Vite.
